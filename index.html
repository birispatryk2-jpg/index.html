SETUP (O SINGURĂ DATĂ)
1 Intră pe workers.cloudflare.com → Sign Up (cont gratuit)
2 Dashboard → Workers & Pages → Create → Create Worker
3 Dă-i un nume (ex: mero-proxy) → Deploy → apoi Edit Code
4 Șterge tot codul existent și lipește codul proxy (vezi fișierul worker.js)
5 Apasă Save and Deploy → copiază URL-ul (ex: mero-proxy.xxx.workers.dev)
6 Lipește URL-ul proxy sus + token-ul Bearer din Mero DevTools
